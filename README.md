# ghostty-dotenv

[Ghostty](https://ghostty.org/) ターミナルエミュレータの設定ファイル。

## ファイル構成

| ファイル | 説明 |
|---|---|
| `config.ghostty` | メイン設定ファイル |

## 主な設定

- **テーマ**: GitHub Dark
- **フォントサイズ**: 9px
- **ウィンドウサイズ**: 150×40

## キーバインド

| キー | 動作 |
|---|---|
| `Alt+Q` | ウィンドウを閉じる |
| `Alt+T` | 新しいタブ |
| `Alt+Enter` | 右に分割 |
| `Alt+Shift+Enter` | 下に分割 |
| `Alt+W` / `Ctrl+W` | ペインを閉じる |
| `Alt+矢印` | ペイン移動 |
| `Alt+Shift+矢印` | ペインリサイズ |
| `Alt+Z` | ペインズーム切り替え |
| `Alt+Shift+;` | フォントサイズ拡大 |
| `Alt+-` | フォントサイズ縮小 |
